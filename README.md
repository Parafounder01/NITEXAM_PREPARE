# ICE HARDMODE — Instrumentation & Control Engineering Practice Quiz

**Original hardest-mode MCQ quiz** for candidates preparing for NIT Tiruchirappalli Technical Assistant recruitment (Advt. No. NITT/R/RC/NT/2026/04), Instrumentation & Control Engineering trade.

Designed for both the **Skill Test** (45 min, qualifying) and **Main Written Test** (120 min, 100 marks).

## Features

- **1010 original multiple-choice questions** across 19 topics at Diploma/UG depth
- **Three modes**: Practice (instant feedback), Exam (timed, feedback at end), Hardest (hard-difficulty pool only)
- **Dark editorial UI** — mobile-first, fully responsive, zero dependencies
- **Persistent progress** via localStorage — resume in-progress sessions
- **Per-topic breakdown** and accuracy tracking
- **Answer review** with correct/incorrect color-coding and explanations
- **Difficulty distribution**: ~70% hard (numerical/application/multi-step), 20% medium, 10% foundational

## How to Run

This is a **zero-dependency static web app**. No build tools, no server, no npm install required.

```
git clone https://github.com/Parafounder01/NITEXAM_PREPARE.git
cd NITEXAM_PREPARE
open index.html
```

Or simply open `index.html` in any modern browser. It works directly from the filesystem (file:// protocol) without CORS issues.
